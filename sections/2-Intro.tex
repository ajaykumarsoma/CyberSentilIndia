\section{Introduction}

Cybercrime in India has grown rapidly, with financial losses exceeding 2.41 lakh crore INR (2020--2024). Traditional rule-based detection systems are inadequate for handling the volume, diversity, and multilingual nature of modern cybercrime, which often involves textual complaints, screenshots, and fraudulent interfaces.

While transformer-based multilingual and multimodal models have advanced NLP and vision-language tasks, existing approaches remain limited to text-only or generic multimodal analysis, lacking specialization for cybercrime detection in multilingual settings. Furthermore, no large-scale annotated dataset exists for the Indian cybercrime landscape.

This paper proposes a transformer-based multilingual multimodal framework integrating textual and visual inputs via cross-modal attention. The approach leverages a 25,000-sample dataset across five Indian languages and a hierarchical taxonomy of 50 cybercrime categories.

The main contributions of this work are as follows:
\begin{itemize}
    \item A novel multilingual multimodal framework for cybercrime detection using transformer-based fusion.
    \item A large-scale, annotated dataset of cybercrime incidents across multiple Indian languages.
    \item A hierarchical and clustered taxonomy comprising 50 cybercrime categories.
    \item Integration of explainable AI techniques to enhance model interpretability.
\end{itemize}
