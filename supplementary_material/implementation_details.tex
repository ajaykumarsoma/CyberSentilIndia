\subsection{Loss Function}

Weighted Cross-Entropy Loss:
\begin{equation}
\mathcal{L} = -\sum_{i=1}^{N} w_i \sum_{c=1}^{C} y_{i,c} \log(\hat{y}_{i,c})
\end{equation}

where $w_i$ is the class weight computed as:
\begin{equation}
w_c = \frac{N}{C \cdot n_c}
\end{equation}

$N$ = total samples, $C$ = number of classes (50), $n_c$ = samples in class $c$.

\section{Data Preprocessing}

\subsection{Text Preprocessing}
\begin{itemize}
\item Tokenization using Qwen2.5 tokenizer
\item Maximum sequence length: 512 tokens
\item Padding strategy: Right padding
\item Truncation: Enabled
\item Special tokens: [CLS], [SEP], [PAD]
\end{itemize}

\subsection{Image Preprocessing}
\begin{itemize}
\item Resize to 224 $\times$ 224 pixels
\item Normalization: ImageNet statistics
\item Mean: [0.485, 0.456, 0.406]
\item Std: [0.229, 0.224, 0.225]
\item Data augmentation: Random horizontal flip (p=0.5)
\end{itemize}
