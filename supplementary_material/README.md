## 📊 Key Highlights

### Taxonomy Coverage
- **20 primary categories** of cybercrime
- **50 fine-grained classes** specific to Indian context
- **Examples** from real cybercrime reports
- **Mapping** to international standards (NIST, ENISA)

### Extended Results
- **Per-category F1 scores** for all 50 classes
- **Confusion matrices** showing error patterns
- **LIME explanations** for 20+ representative samples
- **Attention visualizations** across all 5 languages

### Dataset Details
- **Sample distribution** across languages and categories
- **Annotation agreement** metrics (Fleiss' κ, Cohen's κ)
- **Quality control** procedures
- **Temporal coverage** (2022-2024)
